# Reino Editorial AI Engine — Overview

This system is part of Hebeling OS.

Main modules:

- Editorial Core Engine
- Author Portal
- Staff Dashboard

The system manages the editorial workflow for book publishing.

## Editorial Pipeline

Stages:

1. Ingesta
2. Estructura
3. Estilo
4. Ortotipografía
5. Maquetación
6. Revisión Final

Each project moves sequentially through these stages.

## Core Principle

All modules must reuse the same editorial pipeline logic and database structure.

The Staff Dashboard and Author Portal must remain synchronized.
